# わたしの推し県🗾

-あなたの県の“隠れた良さ”、他県民は気づいてます！-


## サービス概要


「わたしの推し県」は、他県民が「あなたの県の隠れた良さ」を投稿・共有するアプリです。

外からの視点で地元の魅力を再発見し、出身地への肯定感を高めることを目的としています。

褒められた人が、次は誰かの県を褒めたくなる、そのような循環を目指しています。


## このサービスへの思い・作りたい理由


長年「都道府県魅力度ランキング」で最下位常連であった茨城県出身の私は、これまで出身地の良い所、素敵なスポットなどをあまり知ろうともせず「どうせ魅力ないし」と諦めていました。

ところが、他県出身者の方からすると「茨城県は一番住みやすいし、毎週県内で面白そうなところに出かけている。茨城県民が一番魅力を分かっていない」とのこと。「地元には魅力がない」と感じていた私は茨城にも魅力的な部分があるんだと嬉しく思い、実は卑下していただけで地元への愛着はあり、出身地というのは自分のアイデンティティでもあるという事に気づかされました。自分にとってご縁がある場所を他人から認められたことで自己肯定感が上がる経験をしたのです。

このことから、外からの視点で魅力的だなと思う都道府県を「推し県」として語ってもらうといまいち魅力がわかっていない地元の人の心に響くのでは、また地元が好きな方はより地元愛が深まるのでは、と思いこのアプリを考案しました。


## ユーザー層について


**他県民で推し県の魅力を伝えたい人**

- 出身地の人が気づいていない魅力を共有し喜んでほしい
- 旅行や出先での素敵な体験を記録し共有したい
- 応援したい、盛り上げたい都道府県がある

**出身地の魅力が分からない人、知りたい人**

- 出身地のブランド力が弱いと思っている
- 出身地としての自己肯定感が低い
- 地元に興味があり地元愛を深めたい


## サービス利用のイメージ


- 自分の出身県を3件まで登録する(公開/非公開設定あり)
- 投稿者は、出身県以外の都道府県の魅力を「推し県」としての投稿することと、Xでの共有が可能
    - 自分の体験を短文投稿し出身者からのリアクションをもらうことで、推し県を応援する体験をする
    - 都道府県とカテゴリを1つずつ選択し、それについての**好きだと感じた事、嬉しかった体験、自分に刺さった事、自分の県民性との違い、応援したい理由**等を投稿する
    - 【カテゴリ】場所／風景／人／食べ物／暮らし／雰囲気／意外なところ／個人的推しポイント
    - Xで投稿を共有することで、多くの媒体で推し県を主張できる
- 閲覧者は、👀知らなかった／❤いいね／📍行ってみたいのリアクションとコメントが可能
    - 出身県の非公開設定があるためリアクションはカウントのみだが、出身県の人のリアクションが付いた投稿は投稿カードのヘッダーにスター🌟が付き強調される
- スター🌟を獲得した場合にマイページで都道府県のスターバッヂが付与される
- 投稿で地点登録された場所をGoogleマップ上で確認することができる


## ユーザーの獲得について


- 初期投稿で、テストを含め20個程度を投稿することで、ユーザーの投稿のハードルを下げる
- XやMattermost、ソーシャルポートフォリオで宣伝を行い、ユーザーを獲得する


## サービス差別化のポイント・推しポイント


### **競合サービスとの比較**
---
**各種SNS**
- X（旧Twitter）：その時感じたことをテキストと写真で気軽に投稿する。ハッシュタグで関連する投稿を分類することができる。
- LocaLink：地域アカウントでのInstagram情報発信サービス。地域企業がその土地に根差した情報発信を行うことができる。
- ピアッザ：地域密着型のコミュニティアプリ。同じ地域の人とコミュニケーションが取れる。

**差別化ポイント**
各種SNSでは、地域情報以外も含む雑多な投稿の中で特定の地域を関連付けられるものであるか、もしくは特定の地域のみに特化した情報・コミュケーションサービスかに大別される。本アプリでは、各都道府県に関連する情報のみを閲覧・投稿可能とし、出身者からのリアクションが付くことで都道府県を横断したコミュニケーションの可視化ができることが特徴。

---
**旅行・観光情報アプリ**
- Tripadvisor：旅行全般に関する情報や旅行者の口コミをチェックできる。
- 各自治体の観光ガイドアプリ：自治体公式のおすすめ観光スポットの紹介や旅行のシミュレーションができる。
- Googleマップレビュー：登録した地点や店舗のレビューを閲覧・投稿できる。

**差別化ポイント**
旅行・観光情報アプリでは、閲覧側の「失敗したくない」心理から辛口を含む本音レビューが求められるが、本アプリでは他県の魅力を発信することが目的のため、推し県の隠れた魅力や素敵な体験をベースに投稿することで、閲覧側（出身者）に対しては、褒める・魅力を語るなどのポジティブな投稿が集まる。


## 機能候補

**MVP**

- ログイン機能（メールアドレス、パスワード）
- パスワード変更機能
- プロフィール（名前、アイコン画像、出身地(公開/非公開)）
- 推し県投稿機能（CRUD）
    - 都道府県、カテゴリ、推しポイント、場所
- リアクション機能



**本リリース**

- X投稿機能
- Google認証によるログイン機能
- 投稿された場所のGoogle map連携
- 日本地図画像から投稿一覧確認
- リアクション通知機能
- 出身者からのリアクション獲得バッヂ
